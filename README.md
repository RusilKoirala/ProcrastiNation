# 🚀 ProcrastiNation — The Ultimate Productivity Killer

Welcome to the cutting edge of professional idleness. **ProcrastiNation** is an enterprise-grade web application engineered entirely in Next.js to do exactly one thing: absolutely nothing. 

Through state-of-the-art Neobrutalism UI and complex reverse-productivity algorithms, this app ensures you maintain the aesthetic of a hyper-focused tech worker while getting absolutely zero work done.

## 🔮 Core Features

1. **The Infinite Start Button**  
   Click to start working. Navigate through multiple existential confirmation modals. Enjoy an inspirational quote. Watch a loading bar that crashes just before it finishes and reschedules your work for tomorrow. 
   
2. **AI Excuse Generator (ProcrastiAI™ v0.0.1-beta)**  
   Need to explain to your manager why the sprint board hasn't moved? Click a button to generate a highly-professional, highly-absurd excuse. Comes with a Slack `copy-to-clipboard` integration for immediate deployment.
   
3. **Broken Pomodoro Timer**  
   A beautiful, industry-standard Pomodoro clock where the 25-minute "Work" block covertly runs 5x faster than normal time, and your "Break" block counts down 9x slower. Because you've earned it.
   
4. **Reverse Progress Bar**  
   Click "Make Progress" to watch your task completion rise playfully up to 80% before the algorithm begins covertly subtracting progress. Reaching 99% triggers a total meltdown back to 1%.
   
5. **Darkness Embracer**  
   A toggle designed to shift the app into a dark mode. As soon as you flip it, the background transitions smoothly... right past dark mode and into a violently blinding `#ffffff` bright white screen.
   
6. **Achievement Engine**  
   A local state engine that tracks your idleness in real time, serving confetti-pop Sonner toast notifications when you inevitably avoid your responsibilities. 

## 🛠 Tech Stack

Designed to be incredibly overengineered for a joke app:

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v4 
- **Components**: shadcn/ui
- **Aesthetic**: Custom CSS Neobrutalism overrides (`globals.css`)
- **Fonts**: Space Grotesk & Space Mono (loaded natively)
- **Deployment**: Vercel (or just run it locally forever and tell your boss you're "working on the infrastructure")

## 🏃‍♂️ Running Locally

1. Clone the repository so it looks like you're doing open-source work:
   ```bash
   git clone https://github.com/RusilKoirala/ProcrastiNation.git
   ```
2. Delay starting the server by installing packages:
   ```bash
   cd ProcrastiNation
   npm install
   ```
3. Boot the environment and open `http://localhost:3000`:
   ```bash
   npm run dev
   ```
